#  Prime Estates - Real Estate Website

A modern, responsive real estate website built using **HTML, CSS, and JavaScript**.  
This project showcases a clean UI, dropdown navigation, property listings, and basic interactivity — ideal for learning frontend development and deployment with GitHub Pages.

---

##  Live Demo
 https://aditya-varanasi06.github.io

---

##  Features

-  Responsive modern design
-  Navigation bar with dropdown menu
-  Hero section with search bar
-  Featured property listings
-  Contact form
-  Interactive buttons using JavaScript
-  Clean UI with professional color scheme

---

##  Technologies Used

- **HTML5** – Structure of the website  
- **CSS3** – Styling and layout  
- **JavaScript** – Interactivity and functionality
